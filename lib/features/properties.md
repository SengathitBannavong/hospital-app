## This folder using for divide all feature to isolation
### Example:
```
lib/
├── core/            ← Shared code
├── features/
│   ├── auth/        ← Everything about auth in ONE place
│   │   ├── data/
│   │   ├── domain/
│   │   └── presentation/
│   ├── home/        ← Everything about home in ONE place
│   │   ├── data/
│   │   ├── domain/
│   │   └── presentation/
```
